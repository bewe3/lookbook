CSS Zen Garden

MARCH — APRIL 2024

This website was a class group project I participated in for my Advanced CSS class (WDD 331). I was in charge of the header/title and the bottom footer area of the page (all middle content was my classmates). The project requirements were taken from Dave Shea's CSS Zen Garden project, although we didn't officially submit it to the challenge since submissions are closed.
